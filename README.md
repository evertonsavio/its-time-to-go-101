### Golang   
---
* Install: https://golang.org/doc/install?download=go1.16.3.linux-amd64.tar.gz

* Commands
```
go run main.go
go build main.go
GOOS=windows go build main.go
```

* Packages
```
go mod init github.com/havyx/its-time-to-golang
go mod tidy
```

```

check all
```
